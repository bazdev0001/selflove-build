# selflove-build

> Static HTML/CSS design assets for the Lumière Yoga brand — mobile app screen mockups and marketing website templates.

## What It Does

A collection of polished, no-build design assets for a self-love yoga and meditation brand called "Lumière Yoga." The project contains two sets: four mobile app screen mockups (onboarding, class booking, meditation player, progress dashboard) and ten marketing website templates in distinct visual styles. All files are self-contained HTML + CSS with no JavaScript required and no build step.

## Tech Stack

- Language: HTML + CSS (no JavaScript, no build tooling)
- Fonts: Google Fonts (Fraunces serif + Manrope / Inter sans)
- Styling: CSS custom properties shared design system across all screens

## Key Features

- 4 mobile app screens with realistic content (real class names, teachers, GBP pricing — no lorem ipsum)
- 10 website templates covering styles: editorial luxury, organic botanical, dark elegant gold, Swiss typographic, soft glass pastel, zen Japanese, retro 70s, cinematic full-bleed, neo-brutalist, colorblock contemporary
- Shared design palette: plum/rose/terracotta/sage/gold on cream backgrounds
- Phone-framed mockups (390px viewport, dynamic island, status bar) — open `index.html` directly in a browser
- Gallery index page (`templates/index.html`) for browsing all website templates

## How to Run

No build step. Open any screen directly in a browser:

```
apps-templates/01-onboarding/index.html
apps-templates/02-class-booking/index.html
apps-templates/03-meditation-player/index.html
apps-templates/04-progress-dashboard/index.html

templates/index.html   ← gallery of all 10 website styles
```
